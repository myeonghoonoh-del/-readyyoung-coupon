# 🌏✈️ READY YOUNG × LECIRT 콜라보레이션 캠페인

**Travel x Beauty Collaboration** - 모바일 최적화 랜딩페이지

## 🎯 프로젝트 개요

**READY YOUNG** 약국·기초화장품 채널과 **LECIRT** 여행 패키지 채널의 특별 콜라보레이션을 소개하는 모바일 전용 랜딩페이지입니다.

### 🎨 채널 특징 반영
- **READY YOUNG**: 쨍한 블루 컬러, 귀엽고 영한 느낌, 약국·쇼핑·기초화장품 특화
- **LECIRT**: 여행·패키지 특화, 민트 그린 포인트

## ✨ 주요 기능

### 완성된 기능
- ✅ **Hero 섹션**: 두 브랜드 콜라보레이션과 여행×뷰티 컨셉
- ✅ **이벤트 정보 섹션**: LECIRT 여행 패키지 이용자 대상 특전 안내
- ✅ **제품 상세 섹션**: PHARMATIQUE 마스크팩 2종 소개
  - 프리미엄 홍삼 PDRN 리쥬베네이팅 마스크
  - 프리미엄 골드 PDRN 글로우 마스크
- ✅ **매장 위치 섹션**: READY YOUNG 5개 지점 정보
  - 홍대, 명동, 강남, 명동타운, 성수 매장 안내
- ✅ **반응형 디자인**: 모바일 퍼스트, 전 디바이스 대응
- ✅ **애니메이션 효과**: 그라디언트, 플로트, 바운스 애니메이션
- ✅ **이모지 강화**: 귀여운 느낌과 여행 테마 강조

### 디자인 특징
- 🎨 **쨍한 블루 그라디언트**: #0066FF → #00A3FF → #5FDDE5
- 💙 **READY YOUNG 메인 컬러**: 생기 넘치는 블루 계열
- 🌏 **LECIRT 여행 테마**: 비행기, 지구본 아이콘, 민트 포인트
- 💕 **귀여운 핑크 포인트**: #FFB6D9 (강조 요소)
- 📱 **모바일 최적화 레이아웃**
- ⭐ **호버 이펙트와 트랜지션**
